# ZHM-Case-Study

## Overview

This repository contains a case study presentation for Zomato's Healthy Mode feature. The project is implemented as a static HTML page that showcases the user interface and key functionalities of the Healthy Mode, designed to promote healthier food choices on the Zomato platform.

## Features

- **Interactive Hero Section**: Displays a simulated mobile app interface with calorie tracking and healthy food recommendations.
- **Responsive Design**: Built with modern CSS, ensuring compatibility across devices.
- **Visual Elements**: Includes animations, color schemes, and UI components mimicking a real mobile app experience.
- **Case Study Focus**: Highlights how Healthy Mode integrates health-conscious features into food delivery services.

## Technologies Used

- **HTML5**: Structure of the webpage.
- **CSS3**: Styling, animations, and responsive layout.
- **Google Fonts**: Custom fonts for enhanced typography.

## How to View

1. Clone or download the repository.
2. Open `zomato-healthy-mode.html` in any modern web browser.
3. The page will load and display the Healthy Mode case study presentation.

## Project Structure

- `README.md`: This file, providing project documentation.
- `zomato-healthy-mode.html`: The main HTML file containing the case study presentation.
